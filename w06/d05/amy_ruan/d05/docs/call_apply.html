    Call and Apply

  • ¶

    call and apply allow us to invoke a function and specify the context of this.

    var person1 = { name: "Gary", job: "accountant" };
var person2 = { name: "Elaine", job: "teacher" };

say.call(person1);
say.apply(person2);

    Parameters

  • ¶

    the first parameter is the context.

    apply takes an array of paramters for the function as the second parameter

    call takes the parameters inline, after the first context parameter.

    function ifOnly(need, character) {
  return "If I only had " + need + " said the " + character;
}

ifOnly.call(null, "the nerve", "lion");
ifOnly.apply(null, ["a brain", "scarecrow"]);
